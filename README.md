# Bee Decoração e Arte 🐝✨

Bem-vindo ao repositório do site **Bee Decoração e Arte**, um ateliê familiar especializado em amigurumis artesanais. Este projeto foi desenvolvido para oferecer uma experiência digital imersiva, unindo a tradição do artesanato têxtil com uma interface moderna e cinematográfica.

## 🌟 Sobre o Projeto

O site da Bee Decoração e Arte não é apenas um catálogo, mas uma jornada visual. Ele foi construído com o conceito de "Midnight Atelier" (Atmosfera imersiva e emocional) e "Daylight Atelier" (Interface transacional clara e funcional).

### Diferenciais:
- **Atmosfera Global:** Partículas de pólen e poeira de mel que reagem ao mouse.
- **Assistente Bee:** Uma pequena abelha animada que acompanha a navegação.
- **Foco em Conversão:** Integração direta com WhatsApp para encomendas personalizadas.
- **Painel Administrativo Premium:** CMS completo para gestão de produtos, coleções e marketing.

---

## 🛠️ Stack Tecnológica

O projeto utiliza as tecnologias mais modernas do ecossistema Frontend:

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router & Turbopack)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações:** [Framer Motion](https://www.framer.com/motion/)
- **Estado Global:** [Zustand](https://zustand-demo.pmnd.rs/) (com persistência local)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/) & [Radix UI](https://www.radix-ui.com/)
- **Ícones:** [Lucide React](https://lucide.dev/)

## 🚀 Funcionalidades Principais

1. **Catálogo Inteligente:** Filtros em tempo real por categoria e ordenação dinâmica.
2. **Buscador de Presentes:** Um quiz interativo de dois passos para recomendar o amigurumi perfeito.
3. **Lista de Interesse:** Sistema de "carrinho" que formata o pedido e envia diretamente para o WhatsApp do ateliê.
4. **Painel Admin (`/admin`):** 
   - Dashboard com estatísticas.
   - Gestor de produtos com editor modal e preview.
   - Gerenciamento de coleções e mídias.
   - Configurações de marketing (Hero e contatos).

## 🏁 Como Começar

### Pré-requisitos
- Node.js (versão recomendada: 20+)
- NPM, Yarn, PNPM ou Bun

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/beedecor.git
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

O site estará disponível em [http://localhost:3000](http://localhost:3000).
